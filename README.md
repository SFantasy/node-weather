# node-weather

[![Build Status](https://travis-ci.org/SFantasy/send.svg?branch=master)](https://travis-ci.org/SFantasy/node-weather)

在终端中获取天气

## Install

```
[sudo] npm install node-weather [-g]
```

## Usage

```
node-weather 北京
```
